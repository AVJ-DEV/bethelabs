# BETHEL LABS Website

## Installation

1. Place this folder in `c:\xampp\htdocs\appWeb\bethelabs`
2. Start XAMPP (Apache and MySQL)
3. Import the database: Go to phpMyAdmin, create database `bethelabs_db`, and run the SQL from `includes/db.sql`
4. Access via `http://localhost/appWeb/bethelabs`

## Usage

- Navigate through the menu
- Forms submit via PHP (configure mail in php.ini for email sending)
- Add images to `images/` folder
- Update content in PHP files

## Features Added

- Functional contact and inscription forms with validation
- Bootstrap carousel slider on homepage
- Social media sharing buttons
- Enhanced portfolio and news sections
- Responsive design

## Troubleshooting

- Ensure PHP is enabled in XAMPP
- For email, configure SMTP in php.ini
- Responsive design tested on Bootstrap 5
